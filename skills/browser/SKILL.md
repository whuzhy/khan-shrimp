# Browser Skill

## 功能概述

提供浏览器控制能力，让 AI Agent 能够：
- 打开网页并浏览内容
- 截取网页截图
- 模拟点击、输入等用户操作
- 自动化网页任务
- 数据抓取和信息提取

## 使用场景

- 网页信息获取和监控
- 自动化表单填写
- 网页截图和可视化验证
- 数据抓取（需遵守网站规则）
- 网页自动化测试

## 依赖要求

- Chrome 或 Chromium 浏览器
- Playwright 或 Puppeteer 库
- 足够的内存和 CPU 资源

## 注意事项

- 请遵守目标网站的 robots.txt 和使用条款
- 不要用于高频爬取或恶意攻击
- 需要稳定的网络连接
- 某些网站可能有反爬虫机制
