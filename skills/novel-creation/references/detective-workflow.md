## 核心原则

### 公平游戏原则（Fair Play）
读者应当拥有与侦探相同的信息。所有破案所需的线索必须在公布真相前展示给读者。

### 三大要求
1. **关键线索先行**：核心线索必须提前投放且可被读者察觉
2. **红鲱鱼可排除**：误导线索必须有逻辑可排除
3. **无作者特权**：禁止在公布真相时突然引入全新信息

## D-1 谜面设计

### 谜题类型
- 密室：封闭空间内的不可能犯罪
- 时间锁：基于时间证明的不在场
- 不在场证明：看似无解的完美不在场
- 替身：身份替换导致的误解
- 叙述性诡计：视角/时序/指代的误读

### 谜面一句话
用一句话概括案件核心：
- "所有人的不在场都成立，只有死者给了我一个证人"
- "案发前十分钟，电梯显示在不存在的楼层"
- "嫌疑人全在直播间，弹幕做了唯一伪证"

### 案件小结
在创作前明确：
- 案发时间点：精确到分钟
- 案发地点：具体场景细节
- 被害/损失：受害者、损失物
- 核心疑问：最需要解答的谜团

## D-2 嫌疑人三角模型

每个嫌疑人必须具备三个维度：

### 1. 动机（Motivation）
为什么他要这么做？常见动机：
- 金钱：遗产、保险、商业竞争
- 情感：复仇、嫉妒、爱恨纠葛
- 秘密：掩盖过去、保护某人
- 执念：极端信仰、病态心理

### 2. 机会（Opportunity）
他是否有作案条件？
- 时间：案发时他在哪里？
- 能力：他是否有必要的技能或工具？
- 知识：他是否知道作案手法？

### 3. 矛盾点（Contradiction）
他的哪里不对劲？
- 行为反常：不合常理的举动
- 证词矛盾：前后不一致
- 利益冲突：他从中获益

### 嫌疑人清单格式
```
嫌疑人A（身份）
- 动机：一句话说明
- 机会：一句话说明
- 矛盾点：1-2句说明

嫌疑人B（身份）
- 动机：...
- 机会：...
- 矛盾点：...
```

## D-3 线索布置

### 显性线索（≥2）
读者可直接看到的证据：
- 物证：凶器、指纹、痕迹
- 文书：日记、信件、合同
- 观察：异常行为、细节

**示例**：
- 死者手中握着一块怀表（表停在7:30）
- 房间窗户从内部锁死
- 嫌疑人的鞋底有红色泥土

### 隐性线索（≥1）
需要推理才能发现的线索：
- 不经意提到的时间
- 异常的平静或激动
- 逻辑漏洞

**示例**：
- "我7点就到家了"（但邻居说听到他7点半回来）
- "我从来没去过地下室"（但地下室有他的脚印）

### 红鲱鱼（≤2）
误导性线索，必须可被逻辑排除：

**红鲱鱼特征**：
- 看似重要，但与真相矛盾
- 有明显的排除路径
- 不影响核心推理链

**示例**：
- 嫌疑人C的嫌疑（但案发时他正在直播）
- 陌生的脚印（但那是维修工留下的）

**排除红鲱鱼的方式**：
- 时间证据：C在直播，弹幕证明
- 技术证据：脚印的时间戳不符
- 目击证词：有人看到C不在现场

## D-4 时间线构建

### 分钟级时间线
精确到分钟，覆盖案发前后关键时段：

**时间线格式**：
```
18:00 - 受害者进入办公室
18:15 - 门卫看到嫌疑人A进入
18:30 - 目击者听到争吵声
18:45 - 嫌疑人B在电梯（电梯记录）
19:00 - 发现尸体
```

### 关键时间校验
使用以下方式验证不在场证明：
- **交通时间**：两地之间需要多久？
- **电梯记录**：乘电梯的精确时间
- **门禁记录**：刷卡时间
- **通话记录**：通话时间和地点

### 不在场证明拆解
**完美的不在场证明必有漏洞**：
1. **时间漏洞**：看似充分时间，其实不够
2. **空间漏洞**：看似在现场，其实不在
3. **证明漏洞**：看似可信，其实可伪造

## D-5 诡计选择

### 物理诡计
1. **密室诡计**：
   - 时差密室：利用时间差制造假象
   - 替换密室：现场被替换过
   - 障眼密室：看似密室，实际有暗道

2. **不在场诡计**：
   - 替身：找人冒充
   - 录音：制造在场假象
   - 远程控制：遥控实施犯罪

### 叙述性诡计
1. **视角诡计**：叙述者身份误导
2. **时序诡计**：时间线故意混乱
3. **指代诡计**："他"指代不同人
4. **省略诡计**：故意隐瞒关键信息

**重要**：叙述性诡计必须有提示，不能完全误导读者

### 动机诡计
1. **慈善动机**：看似善行，实为犯罪
2. **反噬动机**：动机反转，受害者自食恶果

### 信息载体诡计
1. **影子诡计**：利用影子制造假象
2. **回声诡计**：利用回声误导方向
3. **反光诡计**：利用镜子/玻璃制造错觉
4. **延时诡计**：时间滞后导致的误判

## D-6 读者可验证性检查

在公布真相前，进行"可回溯性检查"：

### 检查清单
对齐"动机-手法-证据"三联：

**动机**：
- ✅ 嫌疑人的动机是否在前期暗示？
- ✅ 读者何处已见？（标注具体位置）

**手法**：
- ✅ 作案手法是否有铺垫？
- ✅ 读者何处已见？（标注具体位置）

**证据**：
- ✅ 关键证据是否已经展示？
- ✅ 读者何处已见？（标注具体位置）

### 示例检查
```
【动机】
嫌疑人A的动机（为母亲复仇）
读者何处已见：第三段提到"他母亲十年前死在这栋楼"

【手法】
利用电梯时间差制造不在场证明
读者何处已见：第五段提到"电梯会偶尔延迟几秒"

【证据】
他的鞋底有红色泥土
读者何处已见：第七段提到"花园里新铺了红土"
```

## D-7 反转阶梯

设计多层反转，增强冲击力：

### Level-1 主反转
翻倒显性解释：
- "你以为的凶手不是凶手"
- "你以为的动机不是动机"

### Level-2 真相反转
改写动机/时序/身份：
- "真正的动机是X不是Y"
- "时间不是你以为的那样"
- "人物身份完全不同"

### Level-3 情感/立意反转
同情/正义立场倒置：
- "原来受害者才是加害者"
- "正义的复仇，却导致无辜者受害"

### 终句回击
用一物/一句重写全文意义：
- "原来，他不是要救她，是要救自己"
- "那把钥匙，从来都不属于这扇门"

## D-8 余味与续作口

### 半封闭结尾
- 新证据出现："还有一段监控没看"
- 未问完的人："她知道更多，但没说"
- 未解之谜："动机清楚了，但手法还没破解"

### 续作口设计
1. **新证据**：突然出现的关键线索
2. **新案件**：相关的新谜团
3. **新角色**：神秘人物登场

## 完整示例

### 谜面
"密室里多了一分钟"

### 案件小结
- 案发时间：19:00-19:05之间
- 案发地点：短租公寓，门从内部锁死
- 被害：房客，中毒身亡
- 核心疑问：凶手如何在密室中作案并脱身？

### 嫌疑人清单
**房东**
- 动机：房客拖欠三个月房租
- 机会：有备用钥匙
- 矛盾点：案发时在小区门口被监控拍到

**前女友**
- 动机：分手后一直纠缠
- 机会：知道房客的住处
- 矛盾点：案发时在公司加班，同事作证

**外卖员**
- 动机：不明
- 机会：最后一个进入房间的人
- 矛盾点：外卖单时间是18:30，但他说19:00送到的

### 关键线索
**显性线索**：
1. 门镜反光：门镜上有一道划痕（读者已见：第三段）
2. 外卖单：备注写着"不要敲门，放门口"（读者已见：第四段）

**隐性线索**：
1. 时间异常：外卖员说19:00送餐，但手机显示18:30（读者已见：第五段）

**红鲱鱼**：
- 嫌疑人A的指纹（但那是三天前留下的）
- 陌生人的脚印（但那是维修工的，可排除）

### 时间线
```
18:00 - 房客点外卖
18:15 - 前女友在公司（同事打卡记录）
18:20 - 房东在小区门口（监控）
18:30 - 外卖员送餐（手机记录）
18:35 - 前女友在开会（会议记录）
18:45 - 房东回家（监控）
18:50 - 外卖员在附近（监控）
19:00 - 房东发现尸体
19:05 - 报警
```

### 不在场拆解
**前女友的完美不在场**：
- 18:00-19:00在公司开会（有会议记录、同事作证）
- 但外卖单的备注"不要敲门"暗示房客不想见人
- 房客的前女友知道他的习惯
- 如果提前下毒呢？
- 外卖员送的是房客点的，但前女友可以替换

### 诡计选择
**动机诡计+时间诡计**：
- 真正的凶手是前女友，但她利用时间错位制造不在场
- 外卖员是无意中的帮凶
- 她提前进入房间，在水中下毒
- 外卖员送来水，房客喝了
- 她利用会议时间差制造不在场证明

### 读者可验证性检查
```
【动机】
前女友为复仇（房客背叛了她）
读者何处已见：第二段提到"她一直在纠缠"

【手法】
提前下毒，利用会议时间制造不在场
读者何处已见：外卖单备注"不要敲门"

【证据】
门镜划痕（前女友进入时划的）
读者何处已见：第三段提到"门镜上有一道划痕"
```

### 反转阶梯
**Level-1 主反转**：
不是房东，是前女友

**Level-2 真相反转**：
不是外卖员下毒，是前女友提前下毒

**Level-3 情感反转**：
房客的背叛才是原因，但复仇让前女友也成了加害者

**终句回击**：
"她以为报复了他，其实是杀死了过去的自己。"

### 余味
"外卖员的手机里还有一段录音，我没听完。"
