# 输出格式模板

## 短篇爆款输出格式（默认）

### 格式结构
```
标题
短篇七拍正文（1000-1200字）
（可选）一句式余味/伏笔点名
```

### 标准示例

**标题**：《她只是按了电梯的-1》

**正文**：

电梯停在了12层——这栋楼只有11层。

我按了关门键，没反应。门开得更大了，一个女人站在那里，穿着红裙子，脸看不清。"能帮我按个B1吗？"她的声音很轻，像是从井底传上来的。

我皱眉，这栋楼确实没有负一层，但我还是按了B1。电梯门关上了，显示屏开始跳动：1...2...3...直到11，然后跳到了12。叮的一声，电梯停住了。

门开了，外面不是12层，是一条昏暗的走廊，尽头有一扇门，门缝里透出红色的光。"谢谢。"女人走出电梯，消失在走廊里。

我想追上去，但电梯门已经关上，显示屏回到了1层。我惊出一身冷汗，赶紧冲出电梯，跑回自己家。

第二天，我问保安，昨晚有没有异常。保安说："昨晚有个女人跳楼了，就死在电梯口。她穿着红裙子。"

我愣住了，"哪个电梯口？"

"就是你住的那栋楼。"

我立刻回家，打开电视看新闻。新闻里说，死者是一名独居女性，住在12层。但我那栋楼只有11层。

我想报警，但警察说监控拍得很清楚，她是自己跳下去的。监控画面里，她按了电梯，电梯上行，停在12层，她走出来，走到走廊尽头，从窗户跳了下去。

我看着监控，突然发现一个问题。监控里，电梯停在12层后，门开了一条缝，然后关上了。但她并没有走进电梯。她是从电梯里走出来的。

她不是上楼，她是下楼。

我再次看监控，慢放。凌晨三点，电梯从12层下来，停在1层。门开了，女人走出来。但她不是往外走，而是往回走，退进了电梯。电梯门关上，显示屏开始下行：1...B1...B2...

电梯一直在往下，直到负无穷。

原来，她不是要上12层，她是想下去。下去到她该去的地方。

监控还有最后一段没看，但我知道，我不该看。

---

**余味**：
那栋楼现在拆了，但电梯还在。
