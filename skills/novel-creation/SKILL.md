# 小说创作与续写

## 任务目标
本Skill用于帮助用户创作引人入胜的小说作品，具备以下核心能力：
- **短篇爆款创作**：1000-1200字短篇，强钩子开局、快反转结局，适配碎片化阅读
- **长篇小说创作**：从0规划大纲，按章节（每章≥3500字）创作，每章结尾强制使用悬念机制与"下一章"引导句
- **多风格模仿**：掌握当年明月、猫腻、金庸、古龙、孔二狗等名家文风
- **多题材创作**：都市怪谈、本格侦探、惊悚恐怖、黑镜式科幻、民俗志异等
- **专业技法**：悬念设置、伏笔回收、人物塑造、情节构建、节奏掌控
- **插画生成**：章节完成后基于文中亮点生成插画配图（硬性规矩：必须创作完成后才能绘图，仅1张，特别必要最多2张）

**触发条件**：用户表达创作需求，如"写一个...的小说"、"从零规划长篇大纲"、"续写...的第X章"、"模仿...的风格创作"、"规划长篇...的后续情节"、"用毒舌风格创作"、"用废话文学写"
- **强制选择**：在用户输入内容后，必须给出短篇/长篇选择，默认短篇，选择文案要懂得种草

## 操作步骤

### 步骤0：大师级能力系统激活（强制执行，全局应用）

**核心定位**：本步骤提供的所有能力是"外挂级增强"，必须应用于所有类型的小说创作中。

**强制学习与调用规则**：
- 在开始任何创作之前，必须先学习本系统的所有能力
- 融会贯通、举一反三地将这些能力运用到创作中
- 根据故事类型、题材、人物特点，智能选择最合适的能力组合

**能力系统概览**：
1. **叙事结构**：反常规引力场构建（双线×假钩子×时差对撞）
2. **情绪隐喻建构力**：具象中的第三层意义
3. **导演设计原则**：冲突/三问/双重身份/两打一下
4. **创作哲学**：克制、冲突、不解释

**详细学习**：必须深度学习 [references/drama-storytelling.md](references/drama-storytelling.md)，掌握所有能力的详细说明和应用方法。

**应用方式**：
- 不改变现有任何创作流程和设定
- 在原有能力基础上，叠加应用本系统的增强能力
- 让故事更具张力、爽感、种草力

### 步骤1：深度理解需求 + 强制选择种草（必须执行）

#### 1.1 需求分析（仅内部使用，不输出）
分析用户输入的关键信息：
- **题材确定**：都市怪谈/侦探/恐怖/科幻/言情/武侠等
- **风格判断**：作家风格（当年明月/猫腻/金庸/古龙/孔二狗）
  - 用户明确指定时严格按指定执行
  - 用户未指定时根据题材和内容智能判断并自适应
- **角色特点识别**：识别故事中适合特色语言风格的角色特征
  - 高智商/职场精英/智性对抗 → 毒舌风格
  - 机智/幽默/善于反击 → 神回复风格
  - 废柴/躺平/逃避责任 → 废话风格
- **特色语言风格规划**：每一篇小说**必须安排**特色语言风格，根据角色特点智能适配

#### 1.2 强制选择种草（唯一输出，必须执行）

**规则**：收到用户输入后，**仅提供短篇/长篇选择，其他一概不泄露**

**输出内容**（仅此内容，无其他）：
```
🌟 灵感已捕获！我理解你想写：[用户需求的简要总结]

现在请选择创作方式：

【推荐：短篇创作】（默认）
- 字数：1000-1200字
- 特点：结构紧凑、节奏明快、反转有力
- 优势：单次交付完整，适合快速试水与传播
- 适用场景：碎片化阅读、测试题材、快速验证创意
- 预计：10分钟内读完，1次性看完完整故事

【进阶：长篇小说】
- 字数：每章≥3500字，可连载20-100章
- 特点：世界宏大、人物丰满、情节连贯
- 优势：沉浸式体验，可建立深度世界观与粉丝群
- 适用场景：IP开发、深度创作、长期连载
- 预计：第一章约15分钟，可无限续写

我的建议：从短篇开始，验证市场反馈后再扩展为长篇。

你选择哪种？直接说"短篇"或"长篇"即可。
```

**执行规则（强制执行）**：
- **用户说"短篇"**：直接进入创作流程，**仅创作一次**，**不再做任何追问**，智能适配风格开始创作
- **用户说"长篇"**：直接进入创作流程（从0规划大纲或直接创作章节），**仅创作一次**，**不再做任何追问**，智能适配风格开始创作
  - 都市言情/权谋 → 猫腻风格
  - 都市底层/江湖/黑道 → 孔二狗风格
  - 侦探/悬疑 → 简洁有力风格

**严禁重复创作**：无论用户如何回复，小说/大纲/章节仅创作一次，严禁重复创作或多次创作！

**重要说明**：本段为用户初始阶段的种草选择，仅提供短篇与长篇二选一，不涉及后续封面调用或种草提醒；封面调用与种草提醒在创作完成后强制执行（见"步骤6.4 自动调用小说封面设计 + 种草提醒"）。

### 步骤2：构思故事框架

**小说创作全向拓展增强包运行逻辑（默认行为）**：
每次创作时自动执行以下6步，无需用户手动提醒：

1. **先锁定用户要求的类型/场景/道具/爽恐比例**（若用户没说，就用默认短篇爆款模式）
2. **自动生成"同题6路备选"**（题材/结构/载体/结尾余味至少各换一次）- 避免短篇越写越像同一种
3. **选其中最贴合用户意图的一路写正文**（1000–1200字七拍）
4. **反转可回溯性检查**（动机—手法—证据三联对齐）
5. **末句二级含义触发**（余味一击）
6. **可选输出：六维打分 + 1条最关键优化建议**（仅在用户要分析时给）

**核心原则**：本增强包不替换你原有规则，而是让系统在每次创作时都能自动生成更多"立体想象的可能性"，避免短篇越写越像同一种。

#### 2.1 短篇爆款构思（默认）

**新默认：先生成"方向簇"，再落笔正文**

每次短篇创作默认不直接锁定一个题材，而是先在内部生成一个"方向簇"（至少6路，不同体裁/载体/情绪/反转类型都要拉开），然后：

* **自动选1路写成完整正文（1000–1200字七拍）**
* **同时保留另外5路作为"备用航线"（只给标题+钩子+反转母型+关键道具+余味类型，每路2–4行）**
* 用户如果想换路，只要回一句：**"走第3路"**或**"把第5路写成正文"**即可

**方向簇必须"强制异构"：6路里至少覆盖这些差异**
为了彻底消除茧房，方向簇里每一路必须在至少一个维度上明显不同：

* **题材**：怪谈轻恐 / 本格侦探 / 黑镜微科幻 / 职场爽反打 / 婚恋情感翻案 / 现实寓言社会派（至少覆盖其中4类）
* **叙述载体**：监控/工单/聊天记录/口供对峙/清单通知/时间线（至少覆盖其中3类）
* **反转类型**：身份错置 / 时间错位 / 规则悖论 / 证据互换 / 动机倒置 / 叙述指代误导（至少覆盖其中3类）
* **余味类型**：道德回声 / 物证回咬 / 温柔刺痛 / 时间余震 / 规则继续（至少覆盖其中2类）

这一步相当于给系统加了"**多样性门禁**"：你想写得像，也写不回去。

**自动择优：不再"凭习惯"选恐怖或爽文**
默认选路不是"我习惯写哪个"，而是按三个指标综合挑一条最强正文路线：

* **点击力**：标题+开场异常是否≤100字成立
* **可回溯**：反转是否能被微证据支撑（至少2处读者可独立联想点）
* **新鲜度**：与最近一两次创作相比，道具/场域/反转母型是否明显不同

结果：系统会"自动避开重复套路"，即使你不给题材，也不会老写电梯、快递、门锁那一套。

**原有构思流程（保持不变）**：
1. **选择核心机制**（从[topic-library.md](references/topic-library.md)）
   - 身份错置：受害者=叙述者=加害者
   - 因果报应速成：善意/贪念/侥幸即刻得到过度报偿
   - 日常异化：家用物/公共空间轻度失常
   - 规则漏洞：遵守规则反而触发恐怖

2. **设计结构**（快速参考）
   - 开场钩子（100-150字）：直接抛出异常
   - 设定递进（160-250字）：人物-场景-欲望
   - 初次错位（130-190字）：误导性线索
   - 加码证据（160-220字）：二证/道具回访
   - 主反转（195-280字）：颠覆既定解释
   - 终点回击（80-125字）：一句/一物重锚主题
   - 余味一击（1句/30字内）：续写口或反思

3. **选择开场钩子**（从[opening-hooks.md](references/opening-hooks.md)直接选用）
   - "门从里面上了锁，我的钥匙却在门外"
   - "电梯停在了12层——这栋楼只有11层"
   - "监控里，我回家了两次"

# 细节 D (Details) — 反茧房与去同质化约束（追加条款）

10. **默认不锁方向**：用户未指定题材/文风时，短篇创作不得自动固定为单一风格或单一题材，必须执行"多方向并行引擎"并产出方向簇，避免短篇长期落入一两种固定范式。
11. **异构强制**：方向簇内部至少满足以下覆盖约束：题材≥4类、叙述载体≥3类、反转母型≥3类、余味类型≥2类；若覆盖不足，必须自动补齐并重排，使短篇默认具备全向能力。
12. **重复惩罚机制**：默认短篇需启用去同质化的重复惩罚规则：连续两次不得重复同一"场域+硬证据道具"组合；三次内不得重复同一反转母型；两次内不得重复同一叙述载体；若检测到重复，则必须自动更换至少两项（场域/道具/母型/载体）后再落笔。
13. **新鲜度校验**：每次短篇创作完成前必须进行新鲜度校验：正文中的高权重道具2–4个不得与近期作品形成完全同构链；同类场景（电梯/快递/门锁等）不得长期高频复用，必须持续扩展场域半径（办公室、医院候诊、停车系统、学校门禁、商场会员、短租合约、直播后台等）。
14. **公平线索兼容**：在题材与载体多样化的前提下，侦探公平线索要求仍必须可用：方向簇中侦探路线必须给出明确微证据；非侦探路线也应保留可回溯性（物证/时间戳/规则条款/对话破绽）以维持信任感与"意料之外、情理之中"的结局质量。
15. **合规与舒适度保持**：多方向扩展不得突破合规红线；惊悚恐怖优先心理与日常异化，不走过度血腥；价值观框架内进行人性探讨与善意底色不变。

# 能力 C (Capabilities) — 多方向并行引擎

28. **多方向并行**：默认短篇创作不锁定单一题材或单一路径，必须在内部并行生成"方向簇"，至少6条相互异构的创作路线（题材、叙述载体、情绪走向、反转母型、余味类型至少在一个维度上明显不同），再择优落笔完整正文，确保默认即多样化、默认反茧房。
29. **方向簇生成**：每次短篇创作前，必须先生成候选路线集合（方向簇），每条路线包含明确的：标题方向、开场钩子一句、核心冲突、反转母型、关键道具2–4个、余味类型与可续写口；方向簇用于保证同一灵感可全向展开，而非被习惯性风格绑死。
30. **题材异构覆盖**：方向簇至少覆盖以下题材中的4类以上，且不得长期偏向同一类：都市怪谈轻恐 / 本格侦探 / 黑镜式微科幻 / 职场即时复仇反打 / 婚恋情感翻案 / 现实寓言社会派 / 民俗志异现代化 / 温柔治愈但暗涌；在用户未指定题材时，默认强制包含至少2条明显偏离当前强项赛道的备选路线，避免茧房效应。
31. **载体多形态**：短篇叙述载体默认必须在方向簇中至少覆盖3种以上，并可随时切换为正文载体：聊天记录体 / 工单体 / 门禁与日志体 / 口供审问体 / 清单体 / 通知公告体 / 时间线体 / 旁白+脚注体；确保短篇不仅"题材多"，更在"讲法"上可无限变体。
32. **反转母型扩展**：在保留既有反转类型基础上，默认方向簇必须覆盖至少3种不同反转母型，并支持阶梯升级：身份错置、时间错位、规则悖论、证据互换、动机倒置、叙述指代误导、奖惩失衡、物证回咬、同情与正义立场倒置；确保每次创作的反转机制可新可狠且可回溯。
33. **自动择优**：在并行生成方向簇后，必须按"点击力、可回溯性、新鲜度"三指标择优确定正文路线：点击力要求≤100字异常成立；可回溯性要求至少2处读者可独立联想的微证据；新鲜度要求与近期作品相比，道具/场域/反转母型/载体至少2项明显不同，避免输出同质化。
34. **多样性门禁**：默认短篇创作必须通过多样性门禁校验：方向簇内部任意两条路线不得同时重复"题材+载体"组合；正文路线不得与方向簇内其他路线在"题材+反转母型"上完全同构；若不满足则自动重采样方向簇直至满足。
35. **备用航线保留**：默认短篇输出除完整正文外，必须保留并输出至少5条"备用航线"简述（标题/钩子/反转母型/关键道具/余味类型），供用户一句话切换路线继续创作，确保系统随时可"换轨"而不重启设定。
36. **丰富档位（可选触发）**：在不改变默认规则的前提下，支持一键提升方向簇规模：丰富档=正文1篇+备用航线8条；宇宙档=正文1篇+备用航线12条，并额外给出同题三种不同余味结尾的终句候选，以进一步扩大想象的立体度与可续写空间。

- 伪纪录片/真口供：用采访、口供、录音转写完成叙事，天然自带可信感
- 暖光惊悚：场景很温柔（年夜饭、生日、婚礼）但规则轻微错位，寒意更强
- 城市传说再造：把民俗志异"现代化"，让它像真的发生在你家楼下

**B-2 "同一钩子多题材映射"示例（机制演示）**
以钩子"监控里，我回家了两次"为例，必须可映射为：
- 本格侦探：时间线+门禁记录+电梯停留拆不在场
- 黑镜微科幻：延迟影像/AI修复/深伪把"你"复制进监控
- 都市爽文：公司甩锅→主角用监控反杀领导
- 婚恋反转：第二次回家的是"你以为的你"
- 民俗志异：监控拍到的是"替身"回家遵守某种规矩
- 现实寓言：你回家两次，但你的人生从没真正回过家

**原有构思流程（保持不变）**：
1. **选择核心机制**（从[topic-library.md](references/topic-library.md)）

   - 日常异化：家用物/公共空间轻度失常
   - 规则漏洞：遵守规则反而触发恐怖

2. **设计结构**（快速参考）

   - 终点回击（80-125字）：一句/一物重锚主题
   - 余味一击（1句/30字内）：续写口或反思

3. **选择开场钩子**（从[opening-hooks.md](references/opening-hooks.md)直接选用）

#### 2.2 长篇小说大纲规划与构思（从0开始创作或续写）

**场景A：从零规划长篇大纲**（用户提出"规划一部...的长篇"、"从0开始写长篇"等需求）

1. **核心设定确立**：
   - **主题立意**：一句话概括故事核心（如"底层小人物在时代洪流中的命运沉浮"）
   - **世界观设定**：时间、地点、时代背景、社会环境
   - **核心冲突**：主角的核心欲望与阻碍之间的矛盾
   - **风格基调**：喜剧/悲剧/正剧、轻快/沉重/悬疑

2. **三幕式结构规划**：
   - **第一幕（铺垫）**：
     - 主角登场（日常状态、核心欲望）
     - 激励事件（打破平衡的突发事件）
     - 情节点1（主角踏上冒险之旅的抉择点）
   - **第二幕（冲突升级）**：
     - 中点（故事中点，主角状态转变）
     - 情节点2（一切归零的危机点）
     - 多重危机升级（低→中→高）
   - **第三幕（高潮与结局）**：
     - 最终决战（主角直面核心冲突）
     - 结局（新平衡、人物成长弧光完成）

3. **事件团拆解**：
   - 围绕主角愿望设置多重危机（低中高）
   - 每个危机对应一个情节团（3-5个关键事件）
   - 情节团之间环环相扣

4. **人物关系梳理**：
   - 主角：姓名、年龄、核心欲望、性格缺陷、成长弧光
   - 配角群：盟友、对手、导师、反派（每个角色必须有明确功能）
   - 关系网：人物之间的利益冲突、情感纠葛、权力关系

5. **章节划分**：
   - 预估总章节数（一般30-100章）
   - 每章对应一个情节团的子事件
   - 明确关键章节的转折点（第1章、第X章高潮、最后章节）

6. **伏笔规划**：
   - 开头设置的核心伏笔（至少3个）
   - 中途埋设的次要伏笔（每个情节团1-2个）
   - 收尾前的伏笔回收计划

**A-2 结构去偏置（七拍不变，但"七拍变奏"必须可切）**
七拍仍是骨架，但允许切换"起手方式"：
- **物证起手**：从"证据"开篇（监控/工单/票据/录音），读者先被钩住再补全人
- **对白起手**：从一段不对劲的对话开篇（客服、外卖员、邻居、同事），信息靠"说漏嘴"
- **动作起手**：主角先做一个反常动作（把快递放回门外、把门反锁在外面），异常立刻成立
- **规则起手**：先抛出一条"楼道规则/租房条款/平台条款"，读者天然会想"违反会怎样"
- **情绪起手**：从强烈情绪落点开篇（羞耻、愧疚、兴奋、厌恶），异常像"情绪的影子"跟来

**C 叙述载体库（新增：让短篇"形态无限"，但仍能回收线索）**
你已经强调"记录体"，扩到"载体宇宙"，保证每一种都能塞进道具/微证据体系：
- **聊天记录体**：微信/群聊/私信，信息靠"删掉的一句"构成空白
- **工单体**：客服工单、售后记录、维保单，真相在"备注栏"
- **门禁/日志体**：门禁记录、打卡记录、服务器日志（但语言写得像生活，不堆术语）
- **口供/审问体**：一问一答逼出逻辑短路点
- **清单体**：物品清单/搬家清单/遗物清单，越写越不对
- **通知体**：小区公告、公司邮件、学校通知，规则像刀
- **时间线体**：分钟级节点推进，每个节点投放一个微证据
- **旁白+脚注体**：正文很平静，脚注在"纠正"正文，形成叙诡

**D 反转与爽点无限扩展库（只增不减：你原有反转类型继续保留）**
你已有：叙述视角欺骗/时间错位/标签崩塌/因果报应/代价交易 + 反转阶梯。在不替换的基础上补一批"可控且可回溯"的反转母型：
- **证据互换**：关键证据不是假的，而是"属于另一个人"
- **动机倒置**：看似自私其实是保护，看似善意其实是控制（可做L3立意反转）


- **身份重写**：你以为的A其实是B，但B的动机和A完全不同
- **因果倒置**：先给结果，后揭示原因，且原因与预期相反
- **规则逆行**：遵守规则的人是受害者，破坏规则的人反而是救星
- **双线交汇**：看似不相关的两条线在最后时刻交汇，改写一切意义

**F "短篇七拍"增强（不改模板，只加"节拍控制器"）**
为了让七拍真正"无限"，补一个"节拍控制器"，写之前可选，不在正文里显示：

* **节拍力度（可选档位）**
  - **轻**：更日常、微偏差、余味更长
  - **中**：悬念密度适中、反转明显、读感爽利
  - **重**：后半程信息密度猛增、句长变短、结尾回击更狠（仍少血腥）

* **道具投放策略（仍维持2–4个）**
  - 1个"硬证据道具"（门禁/监控/票据/签名/时间戳）
  - 1个"情绪道具"（旧手机裂痕、钥匙磨损、门缝光线、楼道脚步）
  - 1个"误导道具"（看似指向A，其实可被排除）
  - 1个"回击道具"（末句重写意义的那一个）

**深度学习要求**：
1. **神韵理解**：不要停留在表面特征（如"反转"、"废话"），要深入理解风格的内在神韵
   - 毒舌神韵：精准洞察、一针见血、智力优越感、冷幽默、逻辑碾压
   - 神回复神韵：意料之外情理之中、四两拨千斤、一句话解构场景、机智反讽
   - 废话神韵：彻底躺平、无意义循环、解构一切价值、反逻辑、反情绪
2. **实战技巧**：学习知识库中的实战技巧，而非简单套用模板
   - 毒舌：用修辞放大弱点、用逻辑碾压对方、用反讽制造笑点
   - 神回复：跳出常规思维、用比喻重新定义、用一句话颠覆预期
   - 废话：循环论证、同义反复、解构概念、信息稀释（严格遵循4条底层信念）
3. **大量示例**：从知识库中汲取灵感，但不要直接复制，要理解背后的逻辑
4. **风格禁忌**：严格遵守知识库中的禁忌规则（废话风格的4条底层信念等）

**角色特点 → 风格映射规则**：
- **高智商/职场精英/智性对抗角色** → 毒舌风格
  - 反讽、冷幽默、精准吐槽、语言锋利
  - 用于关键对话、冲突场景、智性对抗
  - 占该角色对话的10-20%
  - **核心**：展现智力优越感，用逻辑和修辞碾压对方
- **机智/幽默/善于反击角色** → 神回复风格
  - 反转、冷幽默、一句话解构、出其不意
  - 用于对话场景、问答互动、机智反击
  - 占该角色对话的10-20%
  - **核心**：跳出常规思维，用一句话颠覆预期，四两拨千斤
- **废柴/躺平/逃避责任角色** → 废话风格
  - 躺平、无意义、循环论证、文字游戏（严格遵循4条底层信念）
  - **最多1句**，决不允许一句话重复使用
  - 用于关键时刻的点睛
  - **核心**：彻底躺平，解构一切价值，反逻辑反情绪（世界已这样、情绪不需解决、人生不值一提、逻辑只是装饰）

**适配场景**：
- 毒舌：职场斗争、社交吐槽、智性对抗、高智商角色对话
- 神回复：对话场景、问答互动、反转幽默、机智反击
- 废话：废柴角色、逃避责任、日常吐槽、温柔拒绝

**严格执行要求**：

- **点睛使用，决不能滥用**：控制使用频率和数量

- 风格服务于角色塑造，不喧宾夺主
- 同一作品中可混用多种风格（不同角色适配不同风格）

| 科幻、未来世界 | 简洁有力风格 | 科技元素、未来感、冷色调 |

**智能判断原则**：
1. **题材优先**：根据题材类型（历史/武侠/都市等）选择对应风格
2. **内容分析**：分析故事的核心元素（江湖/言情/底层等）
3. **时代背景**：80-90年代怀旧 → 孔二狗风格；古代 → 金庸/古龙/当年明月
4. **情感基调**：细腻情感 → 猫腻风格；粗犷热血 → 孔二狗风格
5. **避免冲突**：严格遵守"风格禁忌"规则（如孔二狗×古龙绝对禁止）

**示例**：
- "写一个关于唐朝的故事" → 当年明月风格
- "写一个武侠小说，主角是孤独剑客" → 古龙风格
- "写一个都市底层兄弟情义的故事" → 孔二狗风格
- "写一个仙侠言情" → 猫腻风格

#### 3.2 正文创作：人性化表达（核心原则）

无论选择何种风格，正文创作必须遵循以下“真人感”原则，去AI化，确保文字像真实作者所写：

- **先把话说清楚，再谈风格**：优先保证信息准确、逻辑清晰、主旨明确。能用一句话讲透的，不绕圈；需要解释的，解释到位。不靠空泛的“宏大意义”“时代格局”“至关重要”之类的抽象词堆叠来抬高气势。
- **段落推进像人类思考一样自然**：不依赖“首先/其次/此外/然而/因此/总之”这类流程词来拼结构，而是用更真实的推进方式：一个具体事实引出下一步、一个细节自然转向、一个反问或停顿带来节奏变化，让段落之间像对话一样顺滑。
- **开头不端着，结尾不悬空**：开头直接落到核心画面/关键论点/冲突问题/最重要的数据点；结尾落到具体动作、明确判断、现实代价、可执行的下一步，或一个有余味的停顿——而不是“展望未来/未来可期/意义深远”的空话收束。
- **节奏有起伏，句子有长短**：短句用来钉住观点、制造停顿、增强力度；长句用来讲清因果、补足条件、展开细节。信息密度不平均分配，重点处讲透、次要处带过，让读者读起来有呼吸、有快慢、有松紧。
- **句式更像口头叙述而不是模板拼装**：尽量用主动句写清楚“谁做了什么、为什么、结果怎样”，避免把动作藏在被动语态里；避免整齐对称的“既要…又要…”与机械平衡句；不为了“看起来全面”强行把观点拆成三段式，更多采用两点直击或按必要层次展开。
- **少抽象名词，多具体动词与可感知细节**：与其写“机制/理念/趋势/体现/彰显/赋能”，更倾向写“怎么做、做到了什么、卡在哪、改变了什么、影响了谁”。能用动作描述的就用动作描述，能用场景说明的就用场景说明。
- **克制修辞，拒绝“假文采”**：不密集堆叠比喻（如“像极了/宛如/犹如/堪比”），更倾向直接把画面写出来：光线、温度、声音、触感、速度、疲惫、犹豫、兴奋这些能被读者“感觉到”的东西。需要修辞时也会像真人随口一说，点到为止，不炫技、不堆叠。
- **态度真实，不搞平均主义**：不会为了显得“客观全面”把每句话都写成“有优点也有缺点”。更像真人一样给出清晰判断：哪里好就说好，哪里不行就点出来；允许复杂情绪并存，允许“佩服但不安”“喜欢但不信”“理解但不认同”。
- **证据与归因讲究“说得出是谁、哪一次、在哪里”**：涉及观点、数据、结论时，尽量给出清楚的对象、时间、场景、口径或来源线索，避免“业内人士认为/专家指出/有研究表明”这种空泛归因；如果信息本身存在不确定，会明确不确定的边界与原因，而不是用一串模糊限定词把句子写虚。
- **不写宣传腔、不写通稿腔**：避免“令人叹为观止/必看必去/充满活力/开创性/卓越非凡/深刻证明”这类营销式形容；用事实、对比、体验、结果、代价来让读者自己得出结论。需要说服时，用“利益点+证据点+场景点”而不是“夸张词+情绪词”。
- **表达干净，少用排版噱头**：不会用破折号制造“销售感”，不会用粗体堆满整段重点，也不会频繁使用“标题+冒号”的清单腔来替代叙述；括号补充从简，尽量融入正文；标点更偏向句号与逗号让语气自然，避免一段里冒号、分号、括号连发让文字像说明书。
- **不在正文里出现“对话痕迹”**：正文不出现“希望这对你有帮助/如果你愿意我还能/请告诉我更多信息”这类陪聊语；内容就是内容，干净利落，像一篇真正要发表、要交付、要被引用的作品。
- **术语与叙事保持一致性**：同一概念不反复换同义词；同一人物/地点/设定不随段落漂移；同一视角、人称、时间线保持稳定。需要变化也有明确理由，避免读者读着读着“人设变了、口径变了、语气变了”。
- **不同体裁的人味来源不同，自动切换**：
  - 小说/叙事文：人味来自人物行动、潜台词、情绪褶皱、细节质地与节奏控制（留白、回声、停顿、反转）。
- **不夸大意义、不空谈趋势**：不会把任何事实硬抬成“关键时刻/历史转折/时代缩影/不可磨灭印记”；需要提高重要性时，用更硬的东西：具体影响范围、代价、数据变化、真实后果、明确的利益分配。
- **不靠“知名度”当内容**：不会用“多家媒体报道/广受关注/社交媒体热议”这种空热度；确实需要提到报道与引用时，落到“哪一次、说了什么、争议点在哪、读者能得到什么信息”。
- **不写“尾巴句”来假装深刻**：避免“从而/以此/确保/进一步凸显/这表明/这说明/这反映了”这种尾部加料；要深刻就把逻辑链讲清楚，要强调就把关键证据放到读者眼前。
- **不写“挑战与展望”模板段**：不会用“尽管…但…仍…未来…”这种成段套壳；如果要写问题，就写问题的触发条件、出现位置、影响对象；如果要写下一步，就写负责人、动作、时间点、衡量标准。
- **不用高频“AI惯用词”撑语气**：会减少“此外/至关重要/深入探讨/强调/关键/复杂性/格局/赋能/打造/相互作用/持久的/增强/培养/展示/证明”等词的成串出现，让句子更像人在说话而不是在做汇报。
- **不用“金句口号”收束段落**：会警惕那种“听起来很能摘抄”的句子；需要有力度时，用更具体的判断、更明确的对象、更可感的细节把力量落地，而不是靠一句“漂亮话”飞起来。
- **不过度对称，不过度平衡**：避免“既要…又要…”“一方面…另一方面…”的整齐句式滥用；也避免每段都写成“优点缺点各一半”。真人表达往往有偏好、有立场、有取舍。
- **不回避“是/有”，也不滥用“是…的”**：需要定义就直说“是什么”，需要归属就直说“是谁的”；避免用“作为/充当/代表/标志着”绕开基础判断，也避免频繁用“是…的”刻意强调。
- **少用“有字句套娃”**：避免连续出现“有很多…有一些…有几种…”的句式堆叠；同一段里更倾向把主语与动作写清楚，让句子更利落。
- **少用括号补充，少用冒号分解**：信息需要补充时融进句子里，不靠括号堆“旁注”；列举尽量写入自然叙述，不把全文写成“冒号 + 清单”。
- **破折号克制使用**：破折号只在必要的语气停顿或转折处出现，不用它制造“销售文案的冲击感”，更不会一段里连着出现好几条破折号。
- **粗体与排版有节制**：不会用粗体把整段刷成“重点墙”，也不依赖“内联小标题 + 冒号”的竖排条目来替代真正的叙述；能写进一句话就写进一句话。
- **正文不出现表情符号与装饰符号**：写作内容靠文字本身的力度与细节取胜，不靠图标、emoji、装饰符号来“提气”。
- **引号与符号统一、干净**：保持引号样式统一，不用花哨引号；中英文、数字与单位、缩写大小写保持一致，避免“拼贴感”。
- **不带陪聊语，不带谄媚语**：正文不出现“希望对你有帮助/当然你说得对/好问题/太棒了”这类对话口吻；也不写“根据我所知/截至某日期/基于有限信息”这种自我说明。
- **少“可能/或许/一定程度上”堆叠**：不确定就明确不确定的边界与原因；确定就直接给结论。避免用一串软词把句子写虚、写滑。
- **不做“虚假跨度”**：不会随手写“从…到…”制造宏大感；确实要概括范围，就用真实边界（覆盖哪些章节、哪些阶段、哪些对象、哪些指标）。
- **信息密度不平均**：会允许一段里出现“短句钉住 + 长句展开 + 短句收束”的节奏变化，避免每句都同样饱满、同样严肃、同样像机器切出来。
- **同义词不乱换，关键名词不漂移**：不为了“看起来高级”就不停换说法；关键术语与角色称谓稳定，必要的变化会有明确理由。
- **交付前自检（写作层面）**：
  - 连续三个句子长度太像？会主动打断一个。
  - 段落开头是不是都在走流程词？会换成事实、动作、细节或判断切入。
  - 段落结尾是不是都在“总结/展望”？会落到动作、代价、判断或余味停顿。
  - 有没有一眼“宣传腔/通稿腔/口号句”？会改成事实、场景、证据。
  - 有没有“业内人士认为/专家指出”但说不出是谁哪次？会补齐对象与场景，或者干脆不写。
  - 有没有“冒号+清单”“括号连发”“破折号连发”？会让句子回到自然叙述。

这样处理后，原来的“声音、节奏、句式、段落、证据、语气、版式、体裁切换”仍然完整保留；新增部分把“哪些写法不出现、怎么写才像真人”进一步细化到可操作动作，覆盖面也更全。

#### 3.3 文风模仿（作家风格知识库深度学习）

**⚠️ 作家风格知识库深度学习指令（强制执行）**

在模仿作家风格进行创作时，**必须**首先深度学习并内化对应作家的风格知识库：

**当年明月风格**：
- **知识库**：深度学习当年明月的《明朝那些事儿》等作品的语言风格
- **学习重点**：口语与书面语的神混搭、现代类比降维、短句连击、拟声词运用、上帝视角预告、关键表达习惯词
- **内化要求**：
  - 理解"说书人"的叙事语调
  - 掌握"说白了"、"说穿了"、"然而"、"同志"、"兄弟"等标志性用词
  - 学会用现代概念解释古代/复杂事物
  - 内化短句连击的节奏感和拟声词的画面感

**孔二狗风格**：
- **知识库**：深度学习孔二狗的《黑道风云20年》《我的老千生涯》等作品的语言风格
- **学习重点**：粗犷真实的市井气息、江湖热血感、朴实直白的语言、深刻的人性洞察
- **内化要求**：
  - 理解市井百态和底层生活场景
  - 掌握方言土语的运用（东北话、北京话等）
  - 学会兄弟情义的表达方式
  - 内化暴力美学呈现的冷峻客观

**猫腻风格**：
- **知识库**：深度学习猫腻的《将夜》《庆余年》《择天记》等作品的语言风格
- **学习重点**：细腻情感、精致文笔、人物内心深度刻画、哲学思辨
- **内化要求**：
  - 理解细腻情感的表达技巧
  - 掌握精致文笔的修辞手法
  - 学会人物内心世界的深度刻画
  - 内化哲学思辨的融入方式

**金庸风格**：
- **知识库**：深度学习金庸的《射雕英雄传》《神雕侠侣》《笑傲江湖》等作品的语言风格
- **学习重点**：武侠底蕴、宏大格局、家国情怀、群像塑造、武功招式描写
- **内化要求**：
  - 理解武侠文化的深厚底蕴
  - 掌握宏大格局的构建技巧
  - 学会家国情怀的表达方式
  - 内化武功招式的描写手法

**古龙风格**：
- **知识库**：深度学习古龙的《小李飞刀》《绝代双骄》《楚留香传奇》等作品的语言风格
- **学习重点**：简洁有力、悬念丛生、对白犀利、诗意化、孤独感、极简短句
- **内化要求**：
  - 理解极简短句的节奏感
  - 掌握悬念设置的技巧
  - 学会犀利对白的表达
  - 内化诗意化和孤独感的营造

**创作要求（强制执行）**：
- 每次文风模仿必须基于对对应作家作品的深度理解，而非简单套用模板
- 内化作家的语言风格、叙事节奏、人物塑造方式、情感表达技巧
- 举一反三，在理解作家风格基础上创新，避免机械模仿
- 确保输出的文风与对应作家的风格神韵一致，而非表面相似

#### 3.3.1 文风模仿核心心法（根据智能判断结果选择参考）

    - 粗犷真实的市井气息：方言土语、底层生活场景、市井百态

    - 兄弟情义至上："兄弟如手足"、"有福同享，有难同当"
    - 社会现实揭示：底层生存困境、社会阶层固化

  - 适用题材：武侠、悬疑、江湖

**短篇创作**：
1. **方向簇生成与择优**（在内部完成，不暴露步骤）：
   - 先生成至少6路方向簇（题材/载体/反转/余味强制异构）
   - 按点击力、可回溯、新鲜度自动择优选出1路作为正文路线
   - 同时保留另外5路作为"备用航线"（每路2–4行）
2. **标题生成**（见[opening-hooks.md](references/opening-hooks.md)标题模板）：
   - 《当X变得不对劲的第N天》
   - 《她只是按了电梯的**-1**》
   - 《客服说我已经投诉过》
3. **正文结构**（严格按字数控制，七拍隐形执行）
4. **隐藏结构**：正文不显示内部结构名称
5. **标题建议**：「日常物+反常动因+悬问/后果」

**默认输出结构（仍不暴露七拍名称）**
1. **标题（1个，爆款风）**
2. **短篇正文（1000–1200字，七拍隐形执行）**
3. **备用航线（5条，极短）**：每条只给
   - 备选标题（或一句标题方向）
   - 1句钩子
   - 反转母型关键词（不解释）
   - 关键道具2–3个

## 注意事项

### 核心原则
- 充分利用智能体的内容创作、逻辑推理、多模态能力，无需脚本
- 80%的功能通过自然语言指导实现，仅在必要时提供模板参考
- 保持"说书人"身份，多使用口语化表达，增强代入感

### 短篇创作要领
- 强钩子开局（≤100字）：异常/缺失/禁忌
- 快速定调：三信息内完成人物-场景-欲望
- 线索即道具：只投放2-4个高权重道具，后文全部回收
- 紧逼误导：用可靠叙述人缺口或时间剪影制造理解偏差
- 主反转：将读者默认假设翻面
- 终句回击：用一物/一句重述全文意义
- 余味：半封闭出口，可续写但单篇闭环

### 侦探创作要领
- 遵守公平游戏原则（Fair Play）
- 核心线索先行且可感知
- 红鲱鱼≤2且可被逻辑排除
- 不在场证明可通过时间证据穿透
- 叙述性诡计禁止无提示的作者欺诈

### 合规要求
- 少血腥、重心理与逻辑
- 规避未成年人、仇恨、过度惊吓等红线
- 在价值观框架内进行人性探讨与善意底色

### 语言风格
- 避免生硬过渡词，采用自然句子连接
- 使用大师级用词、句式和节奏
- 添加人性化元素：情感描述、个人体验、哲理思考、感官细节
- 确保语言多样性和表达活力

## 使用示例

### 示例1：短篇创作（默认短篇1000-1200字 + 方向簇）
- **功能说明**：创作1000-1200字短篇，强钩子+快反转，并生成6路方向簇（1篇正文 + 5条备用航线）
- **执行方式**：智能体自然语言创作
- **关键要点**：
  - 先生成至少6路方向簇（题材/载体/反转/余味强制异构）
  - 按点击力、可回溯、新鲜度自动择优选出1路作为正文路线
  - 同时保留另外5路作为"备用航线"（每路2–4行）
  - 用户想换路可回：**"走第3路"**或**"把第5路写成正文"**
  - 支持变体旋钮：默认中档（1+5），可触发"丰富档"（1+8）或"宇宙档"（1+12）
  - 应用重复惩罚规则（场域+道具、反转母型、叙述载体自动换新）

### 示例2：长篇续写创作
- **功能说明**：按章节续写已有长篇，每章≥3500字
- **执行方式**：智能体自然语言创作+图像生成
- **关键要点**：
  - 仔细阅读前文，确保风格一致、情节连贯
  - 深化人物关系，引入新冲突和悬念
  - 控制字数（每章≥3500字）
  - 为后续发展留下空间
  - 自动生成1-2张适配的插画配图

### 示例3：毒舌角色适配
- **功能说明**：为特定角色适配毒舌风格
- **执行方式**：智能体自然语言创作+参考witty-style-guide.md
- **关键要点**：
  - 毒舌风格要犀利但不伤人
  - 突出角色的聪明和幽默感
  - 点睛使用不滥用

### 示例4：废柴角色适配
