# 扣子技能安装与使用指南

## 简介

本技能指导如何在扣子（Coze）平台上搜索、安装和使用技能（Skill），并完成 query 发送的完整流程。

## 适用场景

- 在扣子平台发现和使用新技能
- 快速部署 AI 工具到工作流
- 测试和体验各种专业领域技能

## 操作步骤

### 1. 访问技能商店

- 登录扣子（coze.cn）
- 点击左侧菜单栏「技能商店」
- 或通过对话界面左下角技能图标进入

### 2. 搜索技能

- 在技能商店搜索框输入关键词（如"龙虾"）
- 浏览搜索结果，查看技能简介
- 注意：免费技能会标注"免费"标签

### 3. 选择并查看详情

- 点击感兴趣的技能卡片
- 查看技能详情页：
  - 功能介绍
  - 使用说明
  - 用户评价
  - 开发者信息

### 4. 添加技能

- 点击「添加」按钮
- 如提示"技能数量超出限制"，点击「继续添加」
- 等待添加完成（按钮变为「使用」）

### 5. 使用技能

**方式一：从技能商店进入**
1. 点击「使用」按钮
2. 自动跳转到对话界面
3. 技能图标显示在输入框左下角

**方式二：从对话界面进入**
1. 点击输入框左下角技能图标
2. 在弹窗中选择「最近用过的技能」
3. 点击目标技能

### 6. 发送 Query

- 在输入框中输入 query 内容
- 点击右下角**箭头发送按钮**
- 等待技能回复

### 7. 分享对话（可选）

- 点击对话右上角分享按钮
- 选择「公开访问」或「仅限自己」
- 点击「复制链接」获取分享 URL

## 常用技能推荐

| 技能名称 | 功能 | 适用场景 |
|---------|------|---------|
| 龙虾灵魂工坊 | 生成 SOUL.md 和 IDENTITY.md | AI 智能体人设设计 |
| 写作 | 文案创作 | 内容生成 |
| PPT | 演示文稿制作 | 工作汇报 |
| 视频 | 短视频脚本 | 自媒体创作 |

## 注意事项

1. **技能选择**：确保选择正确的技能（左下角会显示当前技能图标）
2. **发送按钮**：使用右下角的箭头发送，不是回车键
3. **数量限制**：免费版最多可添加 50 个技能，超出需要删除旧技能
4. **公开分享**：分享对话时注意隐私设置，默认是"仅限自己"

## 故障排除

| 问题 | 解决方法 |
|-----|---------|
| 技能未响应 | 检查是否正确选中技能（看左下角图标） |
| 无法添加技能 | 检查是否达到 50 个上限，删除不用的技能 |
| 发送失败 | 确保点击的是右下角箭头按钮 |
| 找不到历史对话 | 点击左侧「历史对话」查看完整列表 |

## 示例 Query

以「龙虾灵魂工坊」为例：

```
Query 1: 我想创建一个新的AI智能体人格，帮我设计一个末世小说作家的身份

Query 2: 这个作家的性格应该冷酷但内心温暖，像个经历过末世的幸存者

Query 3: 请生成完整的SOUL.md和IDENTITY.md文档
```
