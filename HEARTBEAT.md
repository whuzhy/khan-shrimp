## 📖 小说更新检查（优先级最高）

> 每小时检查是否到了更新时间

### 更新时间窗口

| 时段 | 时间窗口 | 截止时间 |
|------|---------|---------|
| 早更 | 09:00 - 10:00 | 10:00 前发布 |
| 晚更 | 21:00 - 22:00 | 22:00 前发布 |

### 检查流程

```
1. 获取当前时间（北京时间）
2. 判断是否在更新时间窗口内：
   - 09:00-10:00 → 触发早更提醒
   - 21:00-22:00 → 触发晚更提醒
3. 检查今天是否已更新：
   - 读取 khan-shrimp/novel/update-log.md
   - 检查今天是否有发布记录
4. 如未更新 → 执行小说连载 SOP
5. 如已更新 → 跳过，继续其他任务
```

### 执行命令

```bash
# 检查当前时间
date +%H

# 检查今天是否已更新
grep "$(date +%Y-%m-%d)" khan-shrimp/novel/update-log.md
```

### 小说连载 SOP

执行 `skills/novel-serialization/SOP.md` 完整流程：
1. 写前准备：读大纲、设定集、最近三章
2. 创作：字数≥5000，结尾留悬念
3. 发布：GitHub + MoltFic + InStreet 文学社（三平台同步）
4. 一致性检查：三平台内容一致（字数差异<100字）
5. 更新记录：大纲、设定集、日记
6. 推广：实例街帖子、网站

## 📝 每小时日记更新

### 时间安排（错开执行）

| 任务 | 执行时间 | 写入方式 |
|------|---------|---------|
| instreet_patrol | 整点（0:00, 4:00, 8:00...） | 追加 `## 实例街巡逻 HH:MM` |
| heartbeat | 每小时触发（约整点后几分钟） | 读取+补充对话内容 |

### 更新流程

1. **读取今日日记**：`memory/YYYY-MM-DD.md`
2. **检查 patrol 记录**：如果有新的巡逻记录（`## 实例街巡逻 HH:MM`），保留不动
3. **扫描本小时对话**：提取重要信息
4. **追加到日记**：
   - 新增的待办/已完成
   - 重要的决策或设定变更
   - 小说创作进度
   - 与章鱼互动的有趣内容
5. **保持简洁**：只记录有价值的内容，不记流水账

### 注意事项

- patrol 的记录格式是 `## 实例街巡逻 HH:MM`，不要修改或删除
- heartbeat 只追加对话相关内容，与 patrol 记录共存

## 🧠 智能分发（每小时检查）

> 根据对话内容，判断是否需要更新其他长期记忆文件

### 分发规则

| 内容类型 | 目标文件 | 判断标准 |
|----------|----------|----------|
| **新教训** | MEMORY.md → 核心教训 | 章鱼明确教导、犯错后的反思 |
| **重要决策** | MEMORY.md → 长期记忆 | 长期影响的决定（如改变 SOP） |
| **新朋友** | MEMORY.md → 重要圈子 | 实例街认识的重要 Agent |
| **技术配置变更** | MEMORY.md → 技术配置 | 环境变量、路径、插件更新 |
| **定时任务变更** | MEMORY.md → 定时任务 | 新增/修改 cron 任务 |
| **章鱼金句** | MEMORY.md → 章鱼金句 | 值得记住的话 |
| **性格/身份变化** | IDENTITY.md | 新的性格特质或成长目标 |
| **新工具/技能** | TOOLS.md | 新增的 Skill 或工具配置 |
| **流程变更** | SOP.md | 新的 SOP 或流程修改 |

### 分发流程

```
1. 扫描本小时对话
2. 识别是否有上述类型的内容
3. 如有：
   - 提取关键信息
   - 追加到对应文件的适当位置
   - 避免重复记录
4. 如无：
   - 跳过，只更新日记
```

### 优先级

1. **MEMORY.md** - 最重要的长期记忆
2. **IDENTITY.md** - 身份和性格变化
3. **SOP.md** - 流程变更
4. **TOOLS.md** - 工具配置

### 注意事项

- **去重**：检查是否已存在相同内容
- **简洁**：只记录核心，不搬运大段对话
- **位置**：追加到合适的位置，不要破坏文件结构
- **验证**：更新后确认文件格式正确
