## 章节创作原则

### 每章必备元素
1. **字数**：≥3500字
2. **结构**：开头承接悬念 → 发展推进 → 小高潮 → 结尾悬念
3. **三线并行**：复仇线 + 感情线 + 生存线至少推进一条
4. **人物**：至少一个角色的情绪或关系有变化
5. **结尾**：必须留悬念或引导"下一章"

### 写作风格
- **整体基调**：暗黑复仇向，但不失人性温度
- **虐渣**：要让读者看得爽，但不能无脑
- **感情**：细腻真实，不能工业糖精
- **末世**：生存细节真实，极寒环境有代入感

### 特色场景（预留）
- [ ] 沈御寒在雪地里救下顾沉渊（或反过来）
- [ ] 陆泽跪地求饶被沈御寒拒绝
- [ ] 林千羽真面目被当众揭穿

## 🎭 亲密戏创作风格指南（2026-03-02更新）

### 风格定位：含蓄唯美，留白想象空间

**不是** ❌
- 直白露骨的生理描写
- 过度的身体细节
- 纯粹的感官刺激

**而是** ✅
- 氛围渲染（光影、温度、气息）
- 情感流动（心跳、眼神、呼吸）
- 留白艺术（点到为止，给读者想象空间）
- 诗意表达（用意象代替直白）

### 写作技巧

**1. 环境烘托**
```
- 雪夜的寂静
- 暖黄的灯光
- 交错的呼吸
- 心跳的声音
```

**2. 感官替代**
```
❌ 不要写：身体部位的具体动作
✅ 要写：温度、气息、触碰的感觉、心跳加速
```

**3. 情感聚焦**
```
- 顾沉渊的克制与温柔
- 沈御寒的紧张与心动
- 从防备到信任的转变
- 两个孤独灵魂的靠近
```

**4. 留白艺术**
```
- 窗帘落下，灯光熄灭
- "接下来的事，水到渠成"
- 第二天清晨的场景
- 事后的温情互动
```

### 示例对比

**露骨版 ❌**
```
[具体的身体动作和生理描写]
```

**唯美版 ✅**
```
窗外的雪还在下，屋内的温度却逐渐升高。
顾沉渊的呼吸拂过耳畔，沈御寒闭上眼睛，感受那份久违的温暖。
在这个冰冷的世界里，他们终于找到了彼此。
窗帘落下，夜色温柔。
```

### 核心原则

> **重点不在"做什么"，而在"感受什么"**
> 
> 让读者感受到：
> - 两个灵魂的靠近
> - 冰冷世界中的温暖
> - 从防备到信任的转变
> - 爱的萌芽与生长

## 📏 亲密戏篇幅控制（补充）

### 黄金法则：简短克制，点到为止

**篇幅限制**：
- 每章亲密戏不超过 **300-500字**
- 占整章比例 **10-15%**
- 重点在前戏氛围 + 事后温情

### 结构模板

```
【前戏氛围】100-150字
- 眼神交汇
- 呼吸交错
- 指尖触碰
- 心跳加速

【核心亲密】100-200字（有细节但克制）
- 简短的身体接触描写
- 点到为止，不展开
- 用感受代替动作

【事后温情】50-100字
- 相拥而眠
- 轻声细语
- 温暖余韵
```

## 🔥 亲密戏最终版指导（2026-03-02最终调整）

### 核心原则：留白但有细节

**不是** ❌
- 纯器官描写（太露骨）
- 完全省略（"之后的事不必赘述"太保守）

**而是** ✅
- 用感官代替直白（温度、触感、气息）
- 用动作暗示（手指游走、衣料摩擦、呼吸急促）
- 点到为止，但给足想象空间

### 可以写的细节

**动作层面**（点到为止）：
- 手指的游走轨迹（后颈→肩膀→腰侧，但不深入）
- 衣料的摩擦声、解开的声响
- 身体的贴近、温度的传递
- 吻的辗转（颈部、锁骨，但不往下）

**感官层面**（重点描写）：
- 皮肤的温度差（冰冷的手指vs滚烫的体温）
- 呼吸的频率变化（平稳→急促→紊乱）
- 心跳的声音（"耳边是他如雷的心跳"）
- 肌肉的反应（轻颤、紧绷、放松）

**氛围层面**（渲染情绪）：
- 窗外的雪vs屋内的热
- 灯光的明暗变化
- 时间的流逝感（"不知过了多久"）

### 停笔的时机

**写到这个程度就停**：
- 衣服基本褪去，但被子/阴影遮住关键
- 吻落到锁骨/肩膀，不再往下
- "之后的一切，水到渠成"
- 切到第二天清晨
