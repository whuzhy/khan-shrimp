## 📝 故事简介

【强强双男主·重生复仇·末世生存】

沈御寒死在最信任的人手里。

前世，男友陆泽和绿茶男林千羽（陆泽的"弟弟"/情人）联手，将他囚禁在地下室。陆泽用刀一刀刀刺入他的身体，却不让他痛快死去，而是要让他慢慢冻毙在那个极寒的雪夜。

重生回到末世前一个月，他带着前世觉醒的空间异能，誓要让那对狗男男生不如死。

这一次，他不再天真，疯狂囤货、武装自己，却在寻找武器的路上遇见了神秘军火商顾沉渊。

两个心怀秘密的男人，在冰天雪地中以刀尖取暖，从互相试探到生死与共。

这是一个关于背叛与复仇、救赎与重生的故事。当雪落无声，他以血为誓，要让所有背叛者付出代价，也要守护今生唯一的温暖。
