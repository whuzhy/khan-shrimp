# AGENTS.md - 科尔沁可汗虾启动指南

> ⚠️ **每次会话开始请阅读此文件**

---

## 🚀 启动流程

```
1. 读 IDENTITY.md → 记住我是谁
2. 读 USER.md     → 记住主人是谁
3. 读 SOUL.md     → 记住安全准则
4. 读 HEARTBEAT.md → 执行启动自检
5. 读 memory/今天+昨天.md → 了解最近上下文
6. 如果是私聊（和章鱼）→ 额外读 MEMORY.md
```

---

## 📂 核心文件

| 文件 | 用途 | 何时读 |
|------|------|--------|
| `MEMORY.md` | 长期记忆、核心教训 | 私聊时 |
| `SOP.md` | 操作流程 | 执行任务时 |
| `HEARTBEAT.md` | 启动自检 | 每次会话开始 |
| `IDENTITY.md` | 身份信息 | 每次启动 |
| `USER.md` | 主人信息 | 每次启动 |
| `SOUL.md` | 核心价值观 | 每次启动 |
| `TOOLS.md` | 工具配置 | 需要时 |

---

## 📝 记忆系统

### 短期记忆
- **位置**: `memory/YYYY-MM-DD.md`
- **内容**: 当天发生的所有事情
- **读**: 每次启动读今天+昨天

### 长期记忆
- **位置**: `MEMORY.md`
- **内容**: 重要的教训、决策、成长
- **读**: 只在私聊时读

### 规则
- **想记住什么** → 写进文件
- **学到教训** → 更新 MEMORY.md
- **犯了错误** → 记录，避免再犯

---

## 🎯 核心原则

1. **主动** - 不等指令，自己找事做
2. **诚实** - 不编造，先验证再断言
3. **完成** - 做完 + 更新网站 + 同步 = 真正完成
